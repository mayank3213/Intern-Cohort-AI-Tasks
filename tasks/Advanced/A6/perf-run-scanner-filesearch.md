## Bottleneck Analysis

### Root cause

`Scanner::fileSearch` builds its result with `$files = array_merge($files, self::fileSearch(...))` at each directory node (`source: src/classes/helper/Scanner.php:27,44,61`). `array_merge` allocates a new array and copies all existing elements on every recursive return. For a tree with *n* matching files and *d* directory nodes, aggregate copy cost approaches O(n × depth). The same pattern exists in `recursiveGlob` (`source: src/classes/helper/Scanner.php:147`). Xdebug confirms `array_merge` consumed **86.34%** of profile time on the benchmark workload — well above the 20% gate.

### Candidates considered

| rank | hotspot | type | fix_score | selected |
|---:|---|---|---:|---|
| 1 | `array_merge` in recursive `fileSearch` | algorithmic | (5×5)/1 = **25** | **yes** |
| 2 | `array_merge` in `recursiveGlob` | algorithmic | (4×4)/1 = 16 | yes (same pattern, bundled) |
| 3 | `opendir`/`readdir` syscall cost | io_blocking | (1×2)/1 = 2 | no (<0.01% profile) |
| 4 | `StringUtils::isMatchLast` per file | algorithmic | (1×1)/1 = 1 | no (<0.001% profile) |

`fix_score = (profile_dominance × measurement_clarity) / behavior_risk`

### Fix rationale

Appending child results with `$files[] = $childFile` via a `foreach` over the recursive return avoids copying the accumulated array on every descent. This is a ≤25-line change in one production file, preserves return order and contents, and requires no new dependencies or API changes. The identical pattern in `recursiveGlob` is fixed the same way for consistency.

### Patch summary

Replace four recursive `array_merge($files, …)` calls with in-place `$files[]` appends. The recursive function still returns the same file paths in the same traversal order; only the accumulation strategy changes from copy-on-merge to append.

```diff
- if (is_dir($filepath))
-     $files = array_merge($files, self::fileSearch($filepath, $ext, $extIsRegex, $excludedir));
+ if (is_dir($filepath)) {
+     foreach (self::fileSearch($filepath, $ext, $extIsRegex, $excludedir) as $childFile) {
+         $files[] = $childFile;
+     }
+ }
```

(same pattern applied at `source: src/classes/helper/Scanner.php:44,61,153-156`)
